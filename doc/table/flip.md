## State

- `option` Object - 表格配置
 - `key` String - 数据集中的键名
 - `name` String - 表格显示的列名
 - `width` int - 列宽
 - `align` String - 数据在列中的对齐方式
   - `left` - 左对齐
   - `center`- 居中对齐
   - `right` - 右对齐
- `data` Array - 表格数据
